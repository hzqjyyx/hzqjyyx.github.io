如何免费搭建个人博客

Posted on 2017-05-07 | | Views:

备注：必做或强烈推荐阅读的网址，会有标识：
eg： 推荐阅读
本文部分步骤采用直接贴上网址的方式，以压缩篇幅。请多注意

现在有什么方式可以来搭建博客呢

  • 搭建个人服务器，在服务器上安装WordPress，Hexo，Jekyll或者自己实现一套博客系统
    • 优点：可定制性好，能够随意更改behavior，添加更多有趣的功能
    • 缺点：可能大多数时间不是在写博客而是在写代码。。。，付费（很低）
  • WordPress.com，在线服务
    • 优点：可定制性也蛮高的，各种服务，如评论，页面统计都已经做的很好
    • 缺点：但是改起来受到各种条条框框约束
  • Farbox在线服务等
  • 使用Jekyll + Github Pages
  • 使用Hexo + Github Pages

后两者搭建简单，免费，主题丰富，借用Gtihub的服务甚至连CDN都已经帮你做好了。但是这是纯粹的静态页面，很多需要交互的功能都要自己动手添加，比如评论，页面统计等等。

把上面几个通通看了一遍，实在是喜欢Hexo上的NexT主题，界面简洁大方，可定制性高，而且第三方服务集成特别方便，简直是一个完美的主题。

第一开始写博客，暂时还不需要那么多定制功能，先用这个！（其实现在其他平台也已经有仿NexT的主题了，大家可以看看，我就先不尝鲜啦）
