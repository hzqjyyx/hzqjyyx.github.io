VirtualEnv 强烈推荐

VirtualEnv是一个可以让你的电脑上同时存在多个不同版本的Python的工具，人这一生，总会遇到有的项目需要Python3，有的项目只支持Python2，比如奇怪的TensorFlow就支持Python3.5，我当时Python是3.6，难道要为这种事情再买一台电脑么！（当然可以）VirtualEnv在这种情况下就是救星。

VirtualEnv相当于在某个目录下重新安装了一个Python，并且可以把需要安装的库只安装在这个项目里面，保持整台电脑的整洁，可控。

下面介绍一下使用方法

使用

virtualenv my_project will create a folder in the current directory which will contain the Python executable files, and a copy of the pip library which you can use to install other packages. The name of the virtual environment (in this case, it was my_project) can be anything.

virtualenv my_project 会在当前目录下创建一个包含了Python可执行文件, 一份 pip 库的拷贝. 虚拟环境的名字(在此例中，为 my_project) 可以随意命名。

IDE

强烈推荐使用PyCharm

推荐原因：

  1. 有代码高亮，自动补全，跳转，错误提醒等现代IDE该有的功能

  2. 可以使用图形化方式创建VirtualEnv，图形化方式安装库（File-Settings-Project-Project Interpreter)

  3. 项目组织更容易

依赖库

Numpy，Scipy和Matplotlib

Numpy

NumPy is the fundamental package for scientific computing with Python. It contains among other things:

  • a powerful N-dimensional array object
  • sophisticated (broadcasting) functions
  • tools for integrating C/C++ and Fortran code
  • useful linear algebra, Fourier transform, and random number capabilities

Numpy是Python科学计算的基础库，包含了以下功能：

  • 强大的N维数组
  • 完善的broadcasting功能（broadcast真的很厉害）
  • 包含用来集成C/C++和Fortran代码的工具
  • 有用的线性代数，傅里叶变换，随机数等工具

Numpy的安装非常简单

Scipy

The SciPy library is one of the core packages that make up the SciPy stack. It provides many user-friendly and efficient numerical routines such as routines for numerical integration and optimization.

SciPy library是SciPy stack的核心包，提供了很多用户友好，高效的数值计算和优化。

Matplotlib

Matplotlib is a Python 2D plotting library which produces publication quality figures in a variety of hardcopy formats and interactive environments across platforms. Matplotlib can be used in Python scripts, the Python and IPython shell, the jupyter notebook, web application servers, and four graphical user interface toolkits.

Matplotlib是一个Python的2D绘图库，提供了多格式的用于出版质量的图像绘制以及跨平台的交互环境。Matplotlib可以用于Python脚本和IPython shell， the jupyter notebook，web应用服务器，and four graphical user interface toolkits。
