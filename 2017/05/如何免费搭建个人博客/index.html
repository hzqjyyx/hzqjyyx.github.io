如何免费搭建个人博客

Posted on 2017-05-07 |

备注：必做或强烈推荐阅读的网址，会有标识：
eg： 推荐阅读
本文部分步骤采用直接贴上网址的方式，以压缩篇幅。请多注意

现在有什么方式可以来搭建博客呢

  • 搭建个人服务器，在服务器上安装WordPress，Hexo，Jekyll或者自己实现一套博客系统
    • 优点：可定制性好，能够随意更改behavior，添加更多有趣的功能
    • 缺点：可能大多数时间不是在写博客而是在写代码。。。，付费（很低）
  • WordPress.com，在线服务
    • 优点：可定制性也蛮高的，各种服务，如评论，页面统计都已经做的很好
    • 缺点：但是改起来受到各种条条框框约束
  • Farbox在线服务等
  • 使用Jekyll + Github Pages
  • 使用Hexo + Github Pages

后两者搭建简单，免费，主题丰富，借用Gtihub的服务甚至连CDN都已经帮你做好了。但是这是纯粹的静态页面，很多需要交互的功能都要自己动手添加，比如评论，页面统计等等。

把上面几个通通看了一遍，实在是喜欢Hexo上的NexT主题，界面简洁大方，可定制性高，而且第三方服务集成特别方便，简直是一个完美的主题。

第一开始写博客，暂时还不需要那么多定制功能，先用这个！（其实现在其他平台也已经有仿NexT的主题了，大家可以看看，我就先不尝鲜啦）

前提要求

  • 熟练掌握Git基本技能
  • 掌握基本HTML，CSS，JavaScript语言（如果需要修改主题等）
  • 想要一个特别棒的博客

Github Pages

分为项目和个人主页。

个人主页主要是用来做个人网站，每个用户只能有一个这样的repo，项目主页则是对项目的介绍，演示等。

创建方法非常简单，可以参考使用Github Pages建独立博客或者Github Pages官方帮助，按照教程完成后，你应该完成了以下几个部分：

  • 学会使用Git（推荐学习Git使用教程-廖雪峰进行实战）
  • 完成本机Git环境的配置
  • 在Github上注册本机
  • 使用Git命令建立Remote仓库
  • 选择使用User & Project Pages，并新建Repo或者新建pages，具体步骤可以参考如何选择Pages类型-Github Pages帮助

现在，你应该已经准备好Git环境，使用下面的方法确认一下：

  1. 先创建一个index.html文件
  1. 推送到远程仓库
  1. 经过十分钟左右，登录http://yourusername.github.io（个人主页）或者http://username.github.io/repository（项目主页）就可以看到一行Hello World啦。

Hexo

介绍

快速、简洁且高效的博客框架

Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Hexo与Jekyll的选择

Hexo和Jekyll都是使用模板方式编写博客，目的都是帮助我们专心码字，而不是让我们为了改博客而变成前端大神。
Github官方默认的引擎是Jekyll，创建起来非常容易，甚至不需要在本机安装其他程序即可书写博客，因为当你push到Github上以后，会在远程帮你编译，生成合适的网站。
而Hexo，需要Nodejs和npm包管理，不过使用起来也是足够的简单，再加上NexT主题真是个大杀器，最终选择了Hexo。（NexT主题使用参见NexT主题的使用 ）

目录说明

目录 说明
_config.yml 网站的 配置 信息，您可以在此配置大部分的参数。
package.json 安装的应用程序的信息
scaffolds 模版文件夹。当您新建文章时，Hexo 会根据 scaffold 来建立文件。例如，hexo new post Title，则会按照scaffolds文件夹中的post.md生成新文件
source 存放用户资源的地方。除 _posts 文件夹之外，开头命名为 _ (下划线)的文件 / 文件夹和隐藏的文件将会被忽略。Markdown 和 HTML 文件会被解析并放到 public 文件夹，而其他文件会被拷贝过去。
themes 主题文件夹。Hexo 会根据主题来生成静态页面。

图床处理：MPic

博客不可避免的要用到图片，而Github Pages容量只有300m（听说），为了容量和速度，上传到专业的图床可能是更好的选择。国内的话选择七牛云吧，但是每次上传都要同步好再一一复制外链太痛苦。在Windows下，我们有一个神器MPic。只需要把图片往里面推一把，就能自动上传并返回一个图片外链，超级方便。

下载地址：MPic-图床神器

使用方法：设置账号，空间名为七牛对象存储空间名称，AccessKey和SecretKey在七牛个人面板下的密钥管理中，域名为七牛对象存储空间中内容管理页签下的外链默认域名，目录写什么都可以

在Mac，Linux下可以使用qiniu4blog，一个Python库，相当于命令行版的MPic（required python2）
