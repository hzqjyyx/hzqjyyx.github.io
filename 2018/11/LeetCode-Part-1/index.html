1. Two Sum find two elements sum to target. HashMap.

2. Add Two Numbers add two number represented by linked list in reverse order. Iterate one by one with carry. LinkedList Remember to deal with different size of linked list, and leading 1 caused by carry.
3. Longest Substring Without Repeating Characters find the length of longest substring of original string without repeating character. Use array of length 26 to record characters has visited and its index.
Then use Two Pointers. We can expand right pointer as much as possible if there is no same characters. If we find repeated char, which means its index value is not -1(default value), set left pointer to counter[nums[i]]+1.		 HashTable, Two Pointers, String, Substring If they do not specify that characters are restricted to lower letters, then use a map is always a good choice.
Why it is good to use Two Pointers?
Because it is a substring(continues array) problem, and we can use a map to store everything we want.
Why choose to use map and key is char, value is index?
If repeated character comes, all characters between left pointer and the char in previous substring is invalid. So we can store index in order to skip such useless string.
4. Median of Two Sorted Arrays find the median of two sorted array nums1 and nums2. 1. concatenate nums1 and nums2, sort, find median as usual. (fast, easy, memory consumed).
2. make sure nums1 is longer than nums2. split nums1 by i, which also determine j, used to split nums2. if A[i-1]<=B[j] and B[j-1]<=A[i], we are succeed. If not, work like binary search.		 Array, Binary Search, Divide and Conquer The median is splitting array into two equal length subarray.
Corner cases are too difficult to handle.
5. Longest Palindromic Substring find the longest palindromic substring in s 1. DP: Use O(N2)O(N^2)O(N2) space store result of s[i..j]s[i..j]s[i..j]. If s[i..j]s[i..j]s[i..j] is palindromic, and s[i−1]=s[j+1]s[i-1] = s[j+1]s[i−1]=s[j+1], then s[i−1..j+1]s[i-1..j+1]s[i−1..j+1] is palindromic.
2. ExpandAroundCenter: for every element i, expand from s[i] to left and right and from s[i],s[i+1] to left and right until it is not palindromic. It is O(N2)O(N^2)O(N2) time and O(1)O(1)O(1) space.		 String,
DP		 Basic way of 2D DP problem. Fill in diagonal case.
6. ZigZag Conversion display string in zigzag way. find the pattern. every circle is 2*numRows-2 String Hard to classify.
7. Reverse Integer Given a 32-bit signed integer, reverse digits of an integer. get last digit, multiply res by 10, add to res. Math Use long to store interval res. If long is not allowed, compare Integer.MAX_VALUE/10 with res before add.
8. String to Integer (atoi) convert a string to integer follow the specification Math, String think ahead.
space, word, ±, invalid, larger than 32-integer.
9. Palindromic Number determine whether an integer is a palindrome. reverse half of integer, then compare Math reverse until x>reversed x
10. Regular Expression Matching determine whether string matches a pattern. special character are . and * if pattern[i+1]=’*’, process(string[i],pattern[j+2]) or firstMatch & process(string[i+1], pattern[j])
if pattern[i] = string[i], continue.		 String,
DP,
Backtracing		 You have to look ahead. be care of combination of * and .
firstMatch is true when pattern[i] = string[j] or pattern[i] = ‘.’ or i=string.length.
11. Container With Most Water find max area of two vertical lines. start from leftmost and rightmost. move the pointer pointing to the shorter line towards the other end. Array,
Two Pointers		 We can use such a simple method because height of water is determined by lowest line, not by other lines between starting and ending.

String		 easy
14. Longest Common Prefix find longest common prefix string amongst an array of string look one line of chars at a time. If larger than some length or not exactly same for one line, return current prefix. String be careful of the first element, current index may exceed its length.
15. 3Sum find all unique triplet that satisfies a+b+c=0. may contains duplicates sort array. for each element, first, we take it as ‘a’, so b+c = -a. use two pointers. if b+c=-a, then add to result list. if b+c<-a, then increase left pointer, else decrease right pointer
for duplicated nums, only check the first element, and during two pointer search, skip all duplicated nums if we have used one.		 Array
Two Pointers		 in order to keep unique, skip whatever needed.
