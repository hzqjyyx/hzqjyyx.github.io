一维随机变量

概念

随机变量：其值随机会而定的变量，在试验前不能预知其取何值，随机变量就是试验结果的函数。随机事件这个概念实际上就是包容在随机变量这个更广的概念中的。
离散型随机变量：只能取有限个值，或者可列的无限个值。
连续型随机变量：可能的取值不仅无穷多，而且不能无遗漏的列举出，如[−1,1][-1,1][−1,1].
概率分布：全部概率（1）是如何在其可能值之间分配的。
分布函数：表示概率分布的函数，计算“过往的值”。对于离散型随机变量，计算ai<xa_i<xai​<x。对于连续型随机变量，计算X≤xX \leq xX≤x的面积。

F(x)=P(X≤x),x∈(−∞,+∞)\begin{aligned} &F(x)=P(X \leq x), x\in (-\infty,+\infty)\\ \end{aligned} ​F(x)=P(X≤x),x∈(−∞,+∞)​

分布函数的性质：

  • F(X)F(X)F(X)是单调非降：当X1≤x2X_1\leq x_2X1​≤x2​时，有F(X1)≤F(X2)F(X_1)\leq F(X_2)F(X1​)≤F(X2​)
  • 当x→∞x\to \inftyx→∞时，F(x)→1F(x)\to 1F(x)→1；当x→−∞x\to -\inftyx→−∞时，F(X)→0F(X)\to 0F(X)→0
  • P(a≤X≤b)=F(b)−F(a)P(a\leq X \leq b) = F(b)-F(a)P(a≤X≤b)=F(b)−F(a)

离散型随机变量

概率函数与分布函数

概率函数：对于离散型随机变量，每个值的概率共同组成了概率函数。
设X为离散型随机变量，其全部的可能值为a1,a2...{a_1,a_2...}a1​,a2​...，则有XXX的概率函数：

pi=P(X=ai),i=1,2...p_i=P(X=a_i),i=1,2... pi​=P(X=ai​),i=1,2...

分布函数：只对满足条件的ai≤xa_i\leq xai​≤x的那些aia_iai​计算和。

F(x)=P(X≤x)=∑{i∣ai≤x}piF(x)=P(X\leq x)=\sum_{\{i|a_i\leq x\}}p_i F(x)=P(X≤x)={i∣ai​≤x}∑​pi​

连续型随机变量

概率密度与分布函数

概率密度：分布函数的导数称为X的概率密度函数。反映了概率在x点处的密集程度。

F′(x)=f(x)F'(x)=f(x) F′(x)=f(x)

多维随机变量

有时称n维随机变量的分布为：联合分布。

离散型随机变量

概念

设X=(X1,X2...,Xn)X=(X_1,X_2...,X_n)X=(X1​,X2​...,Xn​)为一个n维向量，每个分量X1,X2...X_1,X_2...X1​,X2​...都是一个随机变量，则称X为n维随机变量。
若每个分量XiX_iXi​都是一维离散随机变量，则XXX为离散型。定义离散型随机变量分布时，虽然概念上每个组合都可以有一个概率，但并不需要把每一个组合都列举出来，简单的把未列举出来的设为0即可。

连续型随机变量

概念

设X=(X1,X2...,Xn)X=(X_1,X_2...,X_n)X=(X1​,X2​...,Xn​)为一个n维向量，每个分量X1,X2...X_1,X_2...X1​,X2​...都是一个随机变量。其值可取n维欧氏空间RnR^nRn中的一个点。如果X的全部取值能取满RnR^nRn中的某一区域，则称它为连续的。记{X∈A}\{X\in A\}{X∈A}是一个随机事件，A是RnR^nRn中的一个集合。密度函数为：

P(X∈A)=∫...∫Af(x1,x2...xn)dx1...dxnP(X\in A)=\int...\int\limits_A f(x_1,x_2...x_n)dx_1...dx_n P(X∈A)=∫...A∫​f(x1​,x2​...xn​)dx1​...dxn​

注意：连续型随机变量的实质是有概率密度存在。与离散型不同，连续型不可以简单的定义为各分量都是一维连续型随机变量的那种随机变量。

边缘分布

设X=(X1,...Xn)X=(X_1,...X_n)X=(X1​,...Xn​)为一个n维随机向量。XXX有一定的分布FFF，这是一个n维分布。因为XXX的每个分量XiX_iXi​都是一维随机变量，故他们都有各自的一维分布FiF_iFi​，称为XXX的边缘分布。边缘分布完全由原分布决定。

该表以列表的形式，显示了二维随机变量X=(X1,X2)X=(X_1,X_2)X=(X1​,X2​)的概率分布，我们注意到，X1X_1X1​,X2X_2X2​的分布恰好处在表的边缘，由此得到“边缘”这个名词。

对于离散型分布，一般情况下，固定a1ka_{1k}a1k​，将其他列的全部加起来即可:

P(X1=a1k)=∑j2,...jnp(k,j2,...jn)P(X_1=a_{1k})=\sum_{j_2,...j_n}p(k,j_2,...j_n) P(X1​=a1k​)=j2​,...jn​∑​p(k,j2​,...jn​)

对于连续型分布，设X=(X1,X2)X=(X_1,X_2)X=(X1​,X2​)有概率密度函数f(x1,x2)f(x_1,x_2)f(x1​,x2​)，此时，X1,X2X_1,X_2X1​,X2​分别的概率密度函数为：

f1(x1)=∫−∞+∞f(x1,x2)dx2f2(x2)=∫−∞+∞f(x1,x2)dx1\begin{aligned} f_1(x_1)=\int_{-\infty}^{+\infty}f(x_1,x_2)dx2\\ f_2(x_2)=\int_{-\infty}^{+\infty}f(x_1,x_2)dx1 \end{aligned} f1​(x1​)=∫−∞+∞​f(x1​,x2​)dx2f2​(x2​)=∫−∞+∞​f(x1​,x2​)dx1​

如果X=(X1,X2...Xn)X=(X_1,X_2...X_n)X=(X1​,X2​...Xn​)服从多项分布，则对于X1X_1X1​边缘分布则服从二项分布B∼(N,p1)B\sim (N,p_1)B∼(N,p1​)。
正态分布的边缘分布也为正态分布。

虽然一个随机向量X=(X1,X2...Xn)X=(X_1,X_2...X_n)X=(X1​,X2​...Xn​)的分布FFF足以决定其任意一个分量的边缘分布FiF_iFi​，但反过来并不对：即使知道了所有的XiX_iXi​的边缘分布FiF_iFi​，也不足以决定XXX的分布FFF。因为边缘分布只考虑的单个变量XiX_iXi​的情况，而没有涉及他们之间的关系，而这个信息则是包含在(X1,X2...Xn)(X_1,X_2...X_n)(X1​,X2​...Xn​)的分布之内的。

条件概率与独立性

条件概率

概念

一个随机变量XXX的条件概率分布，就是在某种给定条件下的X的概率分布。设有两个随机变量X1,X2X_1,X_2X1​,X2​，在给定了X2X_2X2​取某个值或某些值的条件下，去求XXX的分布。

例如，考虑一大群人，从其中随机抽取一个，分别以X1X_1X1​和X2X_2X2​记其体重和身高。则X1X_1X1​,X2X_2X2​都是随机变量，它们都有一定的概率分布，现在如限制1.7≤X2≤1.81.7\leq X_2 \leq 1.81.7≤X2​≤1.8(米)，在这个条件下去求X1X_1X1​,的条件分布，这就意味着要从这一大群人中把身高在1.7米和1.8米之间的那些人都挑出来，然后在挑出的人群中求其体重的分布。容易想像，这个分布与不设这个条件的分布（无条件分布）会很不一样。例如，在条件分布中体重取大值的概率会显著增加。

离散型随机变量的条件分布

与第一章所说一样：

P(X2=bj∣X1=ai)=pijP(X1=ai)=pij∑kpik\begin{aligned} P(X_2=b_j|X_1=a_i) &= \frac{p_{ij}}{P(X_1=a_i)}\\ &=\frac{p_{ij}}{\sum_{k}p_{ik}} \end{aligned} P(X2​=bj​∣X1​=ai​)​=P(X1​=ai​)pij​​=∑k​pik​pij​​​

注：推导求了X1X_1X1​的边缘分布。

多项分布：(X1,X2...Xn)(X_1,X_2...X_n)(X1​,X2​...Xn​)服从多项分布M(N;p1,p2...pn)M(N;p_1,p_2...p_n)M(N;p1​,p2​...pn​)，则给定X2=k2X_2=k_2X2​=k2​的条件下，X1X_1X1​的条件分布为二项分布B(N−k2,p1/(1−p2))B(N-k_2,p_1/(1-p_2))B(N−k2​,p1​/(1−p2​))

连续型随机变量的条件分布

设X=(X1,X2)X=(X_1,X_2)X=(X1​,X2​)有概率密度函数f(x1,x2)f(x_1,x_2)f(x1​,x2​)，a≤x2≤ba\leq x_2 \leq ba≤x2​≤b的条件下X1X_1X1​的条件分布的推导如下。

首先求X1X_1X1​的条件分布函数

P(X1≤x1∣a≤X2≤b)=P(X1≤x1,a≤X2≤b)P(a≤X2≤b)P(X_1\leq x_1|a\leq X_2 \leq b)=\frac{P(X_1\leq x_1, a\leq X_2 \leq b)}{P(a\leq X_2 \leq b)} P(X1​≤x1​∣a≤X2​≤b)=P(a≤X2​≤b)P(X1​≤x1​,a≤X2​≤b)​

其中：

P(X1≤x1,a≤X2≤b)=∫−∞x1dt1∫abf(t1,t2)dt2P(a≤X2≤b)=∫abf2(t2)dt2\begin{aligned} &P(X_1\leq x_1, a\leq X_2 \leq b)=\int_{-\infty}^{x_1}\text{d}t_1\int_a^b f(t_1,t_2)\text{d}t_2\\ &P(a\leq X_2 \leq b)=\int_a^bf_2(t_2)\text{d}t_2 \end{aligned} ​P(X1​≤x1​,a≤X2​≤b)=∫−∞x1​​dt1​∫ab​f(t1​,t2​)dt2​P(a≤X2​≤b)=∫ab​f2​(t2​)dt2​​

然后对x1x_1x1​求导，得到条件密度函数

f1(x1∣a≤X2≤b)=∫abf(x1,t2)dt2∫abf2(t2)dt2f_1(x_1|a\leq X_2 \leq b)=\frac{\int_a^bf(x_1,t_2)dt_2}{\int_a^b f_2(t_2)dt_2} f1​(x1​∣a≤X2​≤b)=∫ab​f2​(t2​)dt2​∫ab​f(x1​,t2​)dt2​​

若a=ba=ba=b，则有更简单的形式的条件概率密度:

f1(x1∣x2)=f(x1,x2)f2(x2)f_1(x_1|x_2)=\frac{f(x_1,x_2)}{f_2(x_2)} f1​(x1​∣x2​)=f2​(x2​)f(x1​,x2​)​

也可以写为：

f(x1,x2)=f2(x2)f1(x1∣x2)f(x_1,x_2)=f_2(x_2)f_1(x_1|x_2) f(x1​,x2​)=f2​(x2​)f1​(x1​∣x2​)

也就意味着：两个随机变量X1,X2X_1,X_2X1​,X2​的联合概率密度，等于其中一个变量的概率密度乘以在给定这个变量之下的另一个变量的概率密度。形式和事件中的几乎一致。

将该式两边对x2x_2x2​积分，则有：

f1(x1)=∫−∞+∞f1(x1∣x2)f2(x2)dx2f_1(x_1)=\int_{-\infty}^{+\infty}f_1(x_1|x_2)f_2(x_2)dx_2 f1​(x1​)=∫−∞+∞​f1​(x1​∣x2​)f2​(x2​)dx2​

这个公式可以解释为：X1X_1X1​的无条件概率f1(x1)f_1(x_1)f1​(x1​)是其条件密度f1(x1∣x2)f_1(x_1|x_2)f1​(x1​∣x2​)对条件x2x_2x2​概率的加权平均。该公式也可以看作全概率公式在概率密度下的表现形式。
