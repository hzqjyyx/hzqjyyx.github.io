More general

Something that happens today only affects the output of the system for NNN days into the future. NNN is the width of the system

Yt=f(Xt,Xt−1,...,Xt−N)Y_t=f(X_t, X_{t-1},...,X_{t-N}) Y​t​​=f(X​t​​,X​t−1​​,...,X​t−N​​)

Problem:
For example, in stock, we want to predict not only a few days ago, but also months ago. I mean, long-term trends. Increasing the ‘history’ makes the network more complex which can not be modeled.

NARX network

nonlinear autoregressive network with exogenous inputs

It is a infinite response model.

Basic: What happens today can continue to affect the output forever. In other word, infinite memory.

Yt=f(Xt,Yt−1)Y_t=f(X_t, Y_{t-1}) Y​t​​=f(X​t​​,Y​t−1​​)

(Define: Y−1Y_{-1}Y​−1​​ for t=0t=0t=0)

  • A NARX net with recursion from the output
  • All columns are identical
    • It is only one network, just take a series of input
  • An input at t=0 affects outputs forever

For a generic NARX network, the output YtY_tY​t​​ at time t is computed from

  • past KKK outputs Yt−1,...,Yt−KY_{t-1},...,Y_{t-K}Y​t−1​​,...,Y​t−K​​ and
  • the current and past LLL inputs Xt,...,Xt−LX_t,...,X_{t-L}X​t​​,...,X​t−L​​

That will introduce a problem: the network will explore if we want to ‘remember’ multiple past days.

Simple Recurrent Network

We notice that, memory of the past is in the output itself, and not in the NARX network, which is not good.

We can introduce an explicit memory variable whose job is to remember.

Jordan network and Elman network are not fully recurrent network because their information do not actually flow from end to begin. It is just a ‘early version’ of recurrent network.

Jordan Network

Actually, memory unit simply retains a running average of past outputs. Memory has fixed structure, do not learn to remember.

Elman Network

In Elman network, we separate memory state from output. Use a delay unit, which is a cloned state of hidden layer, and only the weight from the memory unit to the hidden unit is learned.

Why do we need to clone state of hidden layer?

In implementation, once data has been processed in the forward pass, it is erased the connection between hidden unit and memory unit. Therefore, we can treat the series of input as a set of individual input and use default backpropagation to train the model. In this way, the information do not flow from end to begin. I mean, the input at T100T_{100}T​100​​ only flow to T99T_{99}T​99​​ and stop.

How model organized

Minimizing the divergence between the sequence of outputs and the sequence of desired outputs

Given a collection of sequence inputs (Xi,Di)(X_i,D_i)(X​i​​,D​i​​), where Xi=Xi,0,...,Xi,TX_i=X_{i,0},...,X_{i,T}X​i​​=X​i,0​​,...,X​i,T​​, Di=Di,0,...,Di,TD_i=D_{i,0},...,D_{i,T}D​i​​=D​i,0​​,...,D​i,T​​. Train network parameters to minimize the error between the output of the network Yi=Yi,0,...,Yi,TY_i=Y_{i,0},...,Y_{i,T}Y​i​​=Y​i,0​​,...,Y​i,T​​ and the desired outputs. In short, both input and output are matrix. Also, weights, WWW, starts from 0, W(0)W^{(0)}W​(0)​​.

Stability

In general, we do not want the network’s output explore or vanish, in both cases, the network will learn nothing. Bounded means it is not exponential to the input, maybe.

For time-delay structure, if the input X(t)X(t)X(t) is bounded, then the output of the network is also bounded. Because in most cases, activation function will not explore. like ReLU, sigmoid, tanh.

For RNN, things will be different. We can guaranteed the next output is bounded if output and hidden activations are bounded. But will it saturate?

Non-linear System

For scalar non-linearity

  • Sigmoid: Saturates in a limited number of steps, regardless of www
  • Tanh: Sensitive to www, but eventually saturates
    • “Prefers” weights close to 1.0
  • Relu: Sensitive to www, can blow up

For vector non-linearity

Add more ‘tap’

Adding more “taps” adds more “modes” to memory in somewhat non-obvious ways

In summary

  • Bipolar activations (e.g. tanh) have the best behavior
  • Still sensitive to Eigenvalues of WWW
  • Best case memory is short
  • Exponential memory behavior
    • “Forgets” in exponential manner

Jacobian
As we all know, most common activation functions, such as sigmoid, tanh() and RELU have derivatives that are always less than 1. And the derivative of the activation function is always bounded. Then, multiplication by the Jacobian is always a shrinking operation. After a few layers the derivative of the divergence at any time is totally “forgotten”.

Weight
In a single-layer RNN, the weight matrices are identical. The chain product for ∇fkDiv\nabla_{f_k}Div∇​f​k​​​​Div will

  • expand ∇D\nabla D∇D along directions in which the singular values of the weight matrices are greater than 1.
  • shrink ∇D\nabla D∇D in directions where the singular values are less than 1
  • exploding or vanishing gradients
